
The Student Management System is a full-stack web-based application designed to manage student records, courses, and grades. The system includes authentication, a modern dashboard interface, and full CRUD functionality.

## 🚀 Features

- 🔐 Login and Sign-up system
- 👥 Role-based access (Admin, Instructor, Student)
- 🎓 Student Management (CRUD)
- 📚 Course Management (CRUD)
- 📝 Grade Management
- 📊 Dashboard with analytics
- 🔍 Search and filter functionality
- 💾 H2 Database integration

## 🛠️ Technologies Used

- Java
- Spring Boot
- Maven
- H2 Database
- HTML, CSS, JavaScript

## ▶️ How to Run

1. Clone the repository
2. Open in IntelliJ or VS Code
3. Run `StudentManagementSystemApplication.java`
4. Open browser:  
   `http://localhost:8081`
