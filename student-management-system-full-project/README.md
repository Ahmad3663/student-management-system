# Student Management System

This is a simple Spring Boot starter project for COMP2154.

## Features
- Spring Boot web project
- Student controller endpoint
- Basic model classes:
  - Student
  - Course
  - Instructor
  - Enrollment
  - Grade

## How to run
1. Open this project folder in VS Code or IntelliJ.
2. Wait for Maven dependencies to load.
3. Run `StudentSystemApplication.java`
4. Open your browser and go to:
   - `http://localhost:8080/students`

## Expected result
You should see sample student data in JSON format.
