# Student Management System - Merged Nice Version

This version combines the modern sign-in/sign-up screen with the full role-based management system.

## Features
- Modern login + sign-up UI
- Role-based accounts: Admin, Instructor, Student
- Student CRUD
- Course CRUD
- Grade management
- Student dashboard
- H2 database + H2 console

## Demo accounts
- admin / admin123
- instructor / teach123
- student1 / student123

## Sign-up behavior
- Student sign-up creates both a login account and a linked student record
- Admin and Instructor sign-up create management accounts

## Run
Run `StudentManagementSystemApplication.java` and open http://localhost:8081
